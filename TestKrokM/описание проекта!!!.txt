Специализированная программа для подготовки студентов к екзамену КРОКМ (медицина)
http://testcentr.org.ua/index.php/menu-mle/ims/tb-krok-m.html
разработчик программы Брик Олександр Васильович titanum8@ukr.net (студент группы С9П1_26)

Одним из достижений являеться возможности выгрузки в отдельный файл 
только тех вопросов, на которые студент ответил неправильно,
что позволит значит увеличить качество и скорость подготовки студентов.
Количество вариантов ответов неограничено...

Принцип для тестирования с текстового документа.
? в начале строки ------- вопрос
- в начале строки ------- неправильный ответ
+ в начале строки ------- правильный ответ

Рекомендации - Желательно избегать пробелов в начале строки...

Алгоритм тестирования полностью работоспособный.

Доработка данного проекта будет вестись по направлениям:

Отдельный Класс для Тестов (Questions) а в Менеджере List <Questions> 
      с множеством интерактивного доступа к тестам.

Автоматическое распознавание тестов в заданной папке -  их коллекция. 

Возможности создавать тесты с большого набора файлов.

Возможность формировать заданное количество с большого набора вопросов.

Редактор тестов с интерактивным переходом между отдельными блоками.

Возможность определять сложность вопроса по аналитике и выдавать такие вопросы первее.

Формирование теста для студента (группы студентов) по всем его итогам

Сериализация. Сохранение тестов в своем внутреннем зашифрованном формате и работа с ними.

Возможности проводить глубокую аналитику и делать выгрузки рейтинги для студентов и преподавателей.

Создание форм меню для работы с этой программой.

Добавление работы с базой данных (для большого объема данных)

Улучшенная защита проекта (серийники) и инсталятор.

Реализация работы данной програмы по сети.

Интеграция данной системы тестирования на Android платформу.
